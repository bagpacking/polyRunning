# 경로 그리기 앱

Ionic React로 만든 지도 기반 경로 그리기 앱입니다.

## 주요 기능

1. **스플래시 화면**: 앱 시작 시 전체화면 이미지 표시
2. **지도 화면**: OpenStreetMap을 사용한 지도 표시 (완전 무료, API 키 불필요)
3. **경로 그리기**: 손가락으로 지도 위에 경로를 그릴 수 있습니다
   - 한 손가락: 경로 그리기
   - 두 손가락: 줌 인/아웃
   - 줌 후에도 경로 그리기 가능
4. **현재 위치**: 자동으로 현재 위치를 표시합니다
5. **지도 앱 연동**: 확인 버튼을 누르면 그린 경로를 구글 지도 앱으로 전달합니다

## 설치 및 실행

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

**중요**: API 키가 필요 없습니다! OpenStreetMap은 완전 무료입니다.

### 3. 빌드

```bash
npm run build
```

## 사용 방법

1. 앱을 시작하면 스플래시 화면이 표시됩니다 (2초)
2. 지도 화면이 나타나면 현재 위치가 표시됩니다
3. 지도 위에서 손가락으로 경로를 그립니다:
   - **한 손가락으로 그리기**: 지도 위를 손가락으로 드래그하여 경로를 그립니다
   - **두 손가락으로 줌**: 핀치 제스처로 지도를 확대/축소할 수 있습니다
   - **줌 후 그리기**: 줌을 조정한 후에도 계속 경로를 그릴 수 있습니다
4. "지우기" 버튼으로 경로를 초기화할 수 있습니다
5. "확인" 버튼을 누르면 그린 경로가 구글 지도 앱으로 전달됩니다

## 기술 스택

- **Ionic React**: 모바일 앱 프레임워크
- **Leaflet**: 오픈소스 지도 라이브러리
- **OpenStreetMap**: 무료 오픈소스 지도 데이터
- **TypeScript**: 타입 안정성
- **Vite**: 빌드 도구

## 장점

- ✅ **완전 무료**: API 키 불필요, 사용량 제한 없음
- ✅ **오픈소스**: OpenStreetMap과 Leaflet 모두 오픈소스
- ✅ **빠른 로딩**: 가벼운 라이브러리로 빠른 성능
- ✅ **전 세계 커버리지**: OpenStreetMap은 전 세계 지도 데이터 제공

## 주의사항

- 위치 정보 권한이 필요합니다
- 모바일 기기에서 테스트하는 것을 권장합니다
- 구글 지도 앱이 설치되어 있지 않으면 웹 버전으로 열립니다
